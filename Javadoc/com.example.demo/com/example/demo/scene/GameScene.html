Class GameScene

java.lang.Object
com.example.demo.scene.GameScene
public class GameScene extends Object
Handles the main 2048 gameplay scene, including tile movement, scoring, and UI updates.

Responsible for rendering the board, reacting to key events, detecting game over or victory, and triggering restarts or home screen transitions.
