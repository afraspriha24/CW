Class TextMaker

java.lang.Object
com.example.demo.ui.TextMaker
public class TextMaker extends Object
Singleton utility class for creating and managing Text nodes used to display numbers on 2048 game tiles.

Provides standardized styling, positioning, and swapping of Text nodes.
